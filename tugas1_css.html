Assalamu'alaikum warahmatullahi wabarakatuh, Perkenalkan nama saya Muhammad Riyandana Putra, biasa disapa Riyan saya dari Lombok, Nusa Tenggara Barat(NTB), saat ini saya sedang menempuh pendidikan S1 jurusan Teknologi Informasi, semester 7 di UNIVERSITAS BUMIGORA. Sebelumnya Alasan utama saya tertarik mengambil jurusan Teknologi Informasi ini, karena prospek kerjanya sanagat luas pada saat ini dan beberapa tahun kedepannya.

Di kampus saya ada beberapa organisasi atau UKM(Unit Kegiatan Mahasiswa) diantara ada LDK(Lembaga Dakwah Kampus)yang berama UKM LDK KAMIL(Kesatuan Amal Mahasiswa Islam) UKM ini bergerak pada bidang keagamaan khususnya Pada Agama Islam, yang kedua HMTI (Himpunan Mahasiswa Teknologi Informasi) ini adalah perkumpulan mahasiswa yang ada di jurusan Teknologi Informasi. Dan bidang atau keahlian yang saya sedang tekuni saat adalah Web Programming. Kenapa saya tertarik pada bidang ini? Karena melihat perkembangan zaman khususnya didunia kerja, semua perusahan membutuhkan yang mana sebuah website dalam perusahannya.

Pada Semester 7 ini saya mencoba untuk suatu hal yang baru, dimana saya mengambil Mata Kuliah diluar Kampus salah satunya yakni saya mengikuti Program Kampus Merdeka Studi Independen Bersertifikat yang dimana dibawah naungan KEMENDIKBUDRISTEK. dimana disini saya mengambil Kelas Akademi Fullstack Web Developer pada Mitra PT NFCI(Nurul Fikri Cipta Inovasi) atau NF Computer.

Harapan saya mengikuti Program Kampus Merdeka ini adalah untuk memperdalam bidang yang saya tekuni yakni sebagai Web Programmer (Fullstack Web Developer). Dan semoga diakhir kegiatan nanti saya bisa menjadi lebih baik lagi di bidang yang sedang saya tekuni saat ini, dan bisa bermanfaat nantinya didunia pekerjaan.
